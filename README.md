# Simple Servlet Project 🚀

## 📌 Overview
This is a basic Java Servlet project that demonstrates how to:
- Take input from an HTML form
- Send data to a Servlet
- Process the request
- Return a response to the user

This project is ideal for beginners learning Servlets.

---

## 🛠️ Technologies Used
- Java
- Jakarta Servlet (jakarta.servlet)
- HTML
- Apache Tomcat

---

## 📂 Project Structure
SimpleServletProject/

├── src/  
│   └── com/example/servlet/

├── webapp/  
│   ├── index.html  
│   └── WEB-INF/  
│       └── web.xml (optional if using annotations)  

---

## ⚙️ Setup Instructions

### 1. Clone Repository
git clone <your-repo-url>

### 2. Import Project
- Open Eclipse / IntelliJ
- Import as Dynamic Web Project or Maven Project

### 3. Configure Server
- Install Apache Tomcat
- Add project to server

### 4. Run Project
Start Tomcat and open:
http://localhost:8080/SimpleServletProject/
